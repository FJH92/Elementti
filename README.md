
# Elementti

Elementti is a Python-based graphical user interface for processing and visualization of SEM-EDS elemental maps exported in CSV format.

## Features

- import one or multiple CSV map files
- rename imported files
- remove unwanted header rows and columns
- apply file-specific scaling factors
- generate elemental ratio maps
- apply optional masking and conditional preprocessing
- visualize outputs using continuous colormaps or manual bins
- export processed arrays and figure files
- save processing and visualization settings in a JSON summary file

## Source code

The main source code is located in:

`src/elementti.py`

## Requirements

Elementti is implemented in Python and uses:

- PySide6
- NumPy
- Matplotlib
